# FxLedger — Unternehmenskontext

## Firma
- **Name:** FxLedger (Arbeitstitel; vollständiger Firmenname noch offen)
- **Produkt:** MLTraderLog / FxLedger — Handelsjournal für Forex-Trader
- **Phase:** Early-Stage, Produkt in aktiver Entwicklung
- **Gründer:** Einzelgründer

## Produkt-Überblick
- Web-App für Forex-Trader zur Dokumentation und Analyse von Trades
- 4 Themes: Hyper, Linen, Dusk, Auto
- Stack: Node.js Backend, React Frontend, PostgreSQL
- Deployment: Hetzner VPS, Docker

## Strategische Rahmenbedingungen
- Bootstrapped, kein externes Investment
- Fokus auf Qualität vor Wachstum
- Zielgruppe: aktive Retail-Forex-Trader
